#2 — Algebra & Trig Prerequisites

Foundation skills — used in EVERY problem
🚨 These are not "review" — they are tested implicitly in every calculus problem. A sign error or factoring mistake on a limit costs you the whole problem.
Factoring Patterns
a² − b² = (a−b)(a+b)  difference of squares
a³ − b³ = (a−b)(a²+ab+b²)  diff of cubes
a³ + b³ = (a+b)(a²−ab+b²)  sum of cubes
x² + bx + c → find two numbers that multiply to c, add to b
ax² + bx + c → use quadratic formula or factor by grouping
(x+a)² = x² + 2ax + a²  perfect square
🔥 After canceling a factor, ALWAYS state the restriction. (x−2)/(x−2) = 1, x≠2. The professor deducts for missing this on every limit problem. It's an easy point to lose.
Rationalizing (√ in numerator)
When: 0/0 form with a square root. Multiply by conjugate.
(√x − √a)/(x−a) → multiply top & bottom by (√x + √a)
Numerator becomes: x − a
Cancel with denominator factor (x−a) → result has no 0/0.

Example: (√(x+h) − √x)/h → multiply by (√(x+h)+√x):
= ((x+h)−x) / (h·(√(x+h)+√x)) = 1/(√(x+h)+√x)
LCD Method (complex fractions)
When: You have 1/x − 1/a in numerator, divided by (x−a).
Step 1: Combine fractions using LCD = xa:
(1/x − 1/a) = (a−x)/(xa)
Step 2: Divide by (x−a) = multiply by 1/(x−a):
(a−x)/(xa·(x−a)) = −(x−a)/(xa·(x−a)) = −1/(xa)
Step 3: Substitute x=a: −1/(a²)

#3 — Limits — All 7 Techniques

S-TIER 97% probability · 14–20 pts
Algebra/Factoring→ Trig identities→ Graph reading→ LIMITS→ Continuity→ Derivative definition→ Everything in calculus
Concept: lim(x→a) f(x) = L means f(x) gets arbitrarily close to L as x gets close to a — but x never equals a. The limit ignores what happens exactly at x = a.
The 7 Limit Techniques — Decision Order
ALGORITHM: Which technique to use?
1Direct substitution: Plug in x=a. If the result is defined and finite → DONE. Most limits end here.
20/0 form → Factor: Factor numerator and denominator. Cancel common factor. Substitute. ALWAYS state restriction (x≠a).
30/0 with √ → Rationalize: Multiply by conjugate (numerator conjugate if √ is in numerator). This creates a cancelable factor.
40/0 with fractions → LCD: Combine fractions. The combined form will have a cancelable factor.
5Trig 0/0 → Rewrite using identities: Use sin/cos, Pythagorean identities. Target: sinx/x = 1 form.
6One-sided limit (piecewise/|·|): Evaluate from left and right separately using the appropriate piece or sign.
7Bounded × vanishing → Squeeze Theorem: If |f(x)| ≤ (something → 0), use squeeze. (See #5.)
Limits at ±∞
Rational functions: Divide by x^n (highest power in denominator)

Quick rules:
• deg(num) < deg(den) → limit = 0
• deg(num) = deg(den) → limit = ratio of leading coefficients
• deg(num) > deg(den) → limit = ±∞ (check sign)

Polynomial at ±∞: Behavior determined by leading term.
lim(x→∞) 3x³−x+1 = ∞   lim(x→-∞) 3x³ = -∞

For √ expressions at −∞ — THE BIG TRAP:
√(x²) = |x| = x when x>0, but √(x²) = |x| = −x when x<0.
lim(x→−∞) x/√(4x²+1): Factor: x/√(x²(4+1/x²)) = x/(|x|√(4+1/x²))
Since x→−∞: |x|=−x, so = x/(−x·√4) = 1/(−2) = −1/2 ✓

Oblique asymptote check: If deg(num) = deg(den)+1, divide to find oblique asymptote y = mx+b.
Trig Limits
lim(x→0) sin(x)/x = 1 lim(x→0) (1 − cos x)/x = 0 lim(x→0) tan(x)/x = 1 lim(x→0) sin(ax)/bx = a/b lim(x→0) sin(ax)/sin(bx) = a/b lim(x→0) x·cot(x) = 1
PATTERN: Massage to sinθ/θ form If you see sin(3x)/x: multiply top and bottom by 3 → 3·(sin(3x)/3x) → 3·1 = 3
If you see sin(ax)/sin(bx): write as [sin(ax)/ax]·[bx/sin(bx)]·(a/b) → 1·1·(a/b) = a/b
Infinite Limits (→ vertical asymptotes)
If direct sub gives c/0 (c≠0) → limit is ±∞ or DNE. Check sign:
For 1/(x−2) as x→2⁺: numerator=1(+), denominator→0⁺ → +∞
For 1/(x−2) as x→2⁻: numerator=1(+), denominator→0⁻ → −∞
→ lim(x→2) 1/(x−2) = DNE (but lim from each side = ±∞)

Vertical asymptote at x=a if: denominator = 0 at x=a AND numerator ≠ 0 at x=a.
Graph Reading — What to Report
Given a graph, for each x-value a, report:
lim(x→a⁺) f(x): y-value approached from right
lim(x→a⁻) f(x): y-value approached from left
lim(x→a) f(x): exists iff left = right; write the common value
f(a): actual value (closed dot = defined; open dot = undefined)
DNE cases: left ≠ right, or limit → ±∞ (for two-sided limit)
🔥 TRAP: Limit ≠ Function Value lim(x→a) f(x) can exist even when f(a) is undefined. lim(x→2) (x²−4)/(x−2) = 4, even though the function is undefined at x=2. The limit only cares about behavior NEAR x=2, not AT x=2.

#4 — Continuity, IVT & Discontinuities

A-TIER 88% probability · 6–10 pts
3 Conditions of Continuity at x = c
f is continuous at c if ALL THREE hold:
1. f(c) is defined (c is in the domain of f) 2. lim(x→c) f(x) exists (left = right limit) 3. lim(x→c) f(x) = f(c)
☠ EXAM KILLER: Must check and state ALL THREE. Writing only "limit = f(c)" without verifying all three conditions = lost points. Must check them in order.
Types of Discontinuities
TypeWhat happensFix possible?Example
RemovableLimit exists but ≠ f(c), or f(c) is undefined. A "hole."Yes — redefine f(c)(x²−4)/(x−2) at x=2
JumpLeft limit ≠ right limit (both finite)NoFloor function ⌊x⌋ at integers
InfiniteOne or both one-sided limits = ±∞No1/x at x=0, tan(x) at x=π/2
Continuity on Intervals
Continuous on (a,b): Continuous at every point in the open interval.
Continuous on [a,b]: Continuous on (a,b) AND one-sided continuity at endpoints:
→ lim(x→a⁺) f(x) = f(a) AND lim(x→b⁻) f(x) = f(b)

Always continuous (no restrictions needed):
Polynomials · Rational (where defined) · Trig functions (where defined) · Compositions of continuous functions
Intermediate Value Theorem (IVT)
Theorem: If f is continuous on [a,b] and N is any number strictly between f(a) and f(b), then there exists at least one c ∈ (a,b) such that f(c) = N.

Required conditions: f continuous on CLOSED interval [a,b]. N strictly between the two output values.

Most common exam use: Show that f(x) = k has a solution, i.e., "a root exists."
ALGORITHM: IVT to show a root exists
1Define g(x) = f(x) − target value (e.g., if showing f(c)=5, let g(x)=f(x)−5)
2State that g is continuous on [a,b] (and why — polynomial, composition, etc.)
3Compute g(a) and g(b). Show they have opposite signs.
4Invoke: "Since g is continuous on [a,b] and g(a) < 0 < g(b) [or reverse], by the IVT, there exists c ∈ (a,b) such that g(c) = 0."
5Translate back: "Thus f(c) = target value." ✓
EXAM STYLE — SG1 Q18: Show x⁴+x−3=0 has a root on (1,2)
Letg(x) = x⁴+x−3 (polynomial → continuous on [1,2])
g(1)= 1+1−3 = −1 < 0
g(2)= 16+2−3 = 15 > 0
IVTSince g continuous, g(1)<0<g(2), by IVT ∃c∈(1,2) with g(c)=0 → x⁴+x−3=0 ✓
Exam-Style Continuity Analysis of Piecewise Function
EXAM STYLE — FR Q4: Find all x where f is continuous, all x where differentiable
f(x) = {3 if x<−2 ; 1/(x−1) if |x|≤2, x≠1 ; 1 if x>2}

Check x = −2: f(−2) = 1/(−2−1) = −1/3. lim⁻ = 3. lim⁺ = −1/3. lim⁻ ≠ lim⁺ → jump discontinuity at x=−2
Check x = 1: f(1) undefined (excluded). lim⁻ = −∞, lim⁺ = +∞ → infinite discontinuity at x=1
Check x = 2: f(2) = 1/(2−1) = 1. lim⁻ = 1/(2−1) = 1. lim⁺ = 1. All three conditions ✓ → continuous at x=2
Continuity: f is continuous on (−∞,−2) ∪ (−2,1) ∪ (1,2] ∪ (2,∞) = (−∞,−2) ∪ (−2,1) ∪ (1,∞)
Differentiability: Subset of continuity (also check corners/cusps where defined). At x=2: check one-sided derivatives.
MEMORY: "Continuous First, Differentiable Second" Differentiable ⟹ Continuous (if f is differentiable at a, it is continuous there). But Continuous ⟹ NOT necessarily differentiable (|x| at 0 is continuous but not differentiable). So: differentiability ⊂ continuity.
30-SECOND VERSION: Continuity at c (1) f(c) defined? (2) Both one-sided limits exist and are equal? (3) That common value = f(c)? If ALL YES → continuous. If ANY NO → state which condition fails and classify the discontinuity type.

#5 — Epsilon-Delta & Squeeze Theorem

B/C-TIER 75–78% probability · 4–6 pts
Epsilon-Delta Definition
Formal definition:
lim(x→a) f(x) = L means: for every ε > 0, there exists δ > 0 such that:
if 0 < |x − a| < δ, then |f(x) − L| < ε
STEP 1 ALGORITHM: Finding δ (what is always asked)
1Write |f(x) − L| and substitute f(x).
2Simplify algebraically until you get an expression of the form: C·|x − a| where C is a constant.
3Set C·|x−a| < ε → |x−a| < ε/C.
4Therefore δ = ε/C. (Sometimes δ = min(1, ε/C) if needed.)
EXAM STYLE — FR Q1: lim(x→4) (2x/5 + 1) = 13/5
|f(x)−L||(2x/5+1) − 13/5| = |2x/5 − 8/5| = (2/5)|x−4|
Need(2/5)|x−4| < ε
So|x−4| < 5ε/2
δ =5ε/2
Write"Let δ = 5ε/2. If 0<|x−4|<δ, then |f(x)−13/5| = (2/5)|x−4| < (2/5)·(5ε/2) = ε. ✓"
Squeeze Theorem
Theorem: If h(x) ≤ f(x) ≤ g(x) near a (but not necessarily at a),
and lim(x→a) h(x) = lim(x→a) g(x) = L,
then lim(x→a) f(x) = L
ALGORITHM: Squeeze Theorem Proof
1Identify the bounded part: usually a trig function. Write: −1 ≤ sin(anything) ≤ 1.
2Multiply all sides by the other factor. CRUCIAL: If multiplying by x² (always ≥ 0), inequality direction unchanged. If multiplying by a possibly-negative quantity, handle signs carefully.
3Take lim(x→a) of both outer functions. Both should → L.
4Conclude: "By the Squeeze Theorem, lim f(x) = L." State the theorem by name.
EXAM STYLE — Exam 1 Q2: Prove lim(x→0) x²sin(1/x²) = 0
Step 1−1 ≤ sin(1/x²) ≤ 1 for all x ≠ 0
Step 2Multiply by x² ≥ 0: −x² ≤ x²sin(1/x²) ≤ x²
Step 3lim(x→0) −x² = 0 AND lim(x→0) x² = 0
ConcludeBy the Squeeze Theorem, lim(x→0) x²sin(1/x²) = 0. ✓
TrapMust state "multiply by x² ≥ 0" to justify direction of inequality
MEMORY: Squeeze Theorem = Sandwich Theorem Picture f(x) squeezed between h(x) below and g(x) above. If top and bottom bread both go to L, the filling must also go to L. Classic targets: anything × sin(1/x), anything × cos(anything) where the "anything" → 0.

#6 — Derivatives — All Rules

S-TIER 95% probability · 12–18 pts
Limits→ Difference quotient→ DERIVATIVES→ Critical numbers→ Sign charts→ Extrema→ Optimization
Derivative Definition
f′(x) = lim(h→0) [f(x+h) − f(x)] / h f′(a) = lim(x→a) [f(x) − f(a)] / (x − a)
Geometric meaning: Slope of the tangent line to y=f(x) at each point.
Physical meaning: Instantaneous rate of change of f at x.
When does f′(a) fail to exist? Corner/cusp (slopes disagree), vertical tangent (slope = ±∞), or discontinuity.
All Differentiation Rules
d/dx[c] = 0
d/dx[xⁿ] = nxⁿ⁻¹  (Power Rule — works for ALL real n)
d/dx[cf(x)] = c·f′(x)  (Constant Multiple)
d/dx[f±g] = f′ ± g′  (Sum/Difference)
d/dx[fg] = f′g + fg′  (Product Rule)
d/dx[f/g] = (f′g − fg′)/g²  (Quotient Rule)
d/dx[f(g(x))] = f′(g(x))·g′(x)  (Chain Rule)
Trig Derivatives — All 6
d/dx(sin x) = cos x
d/dx(cos x) = −sin x
d/dx(tan x) = sec²x
d/dx(cot x) = −csc²x
d/dx(sec x) = sec x·tan x
d/dx(csc x) = −csc x·cot x
MEMORY: Co-functions ALL get a minus sign cos, cot, csc all have negative derivatives. sin→cos, tan→sec², sec→sec·tan. The "co" versions: cos→−sin, cot→−csc², csc→−csc·cot.
Chain Rule — Deep Dive
"Outside then inside": Derivative of outer function evaluated at inner, times derivative of inner.

Single layer: d/dx[sin(3x)] = cos(3x)·3
Two layers: d/dx[sin²(3x)] = 2sin(3x)·cos(3x)·3
Three layers: d/dx[sin(tan(√(1+x³)))]
  = cos(tan(√(1+x³))) · sec²(√(1+x³)) · (1/(2√(1+x³))) · 3x²
🔥 MOST COMMON TRAP: Forgetting the inner derivative d/dx[sin(3x)] = cos(3x)·3, NOT just cos(3x). d/dx[(x²+1)⁵] = 5(x²+1)⁴·2x, NOT 5(x²+1)⁴.
FTC Part 2 — Derivative of Integral
d/dx[∫ₐˣ f(t)dt] = f(x) d/dx[∫ₐ^(g(x)) f(t)dt] = f(g(x))·g′(x) d/dx[∫ₓᵇ f(t)dt] = −f(x)
EXAM 4 Q1 (8 pts): f(x) = ∫₂^(cosx) 1/(1−t²) dt, find f′(x)
SetupVariable upper bound → FTC2 + chain rule
FTC2f′(x) = 1/(1−cos²x)·(−sinx)
Simplify1−cos²x = sin²x (Pythagorean identity)
Answer= −sinx/sin²x = −1/sinx = −cscx
TrapMissing the (−sinx) chain rule factor = ~5 pts lost
☠ EXAM KILLER: FTC2 Chain Rule Factor d/dx[∫ₐ^(cosx) f(t)dt] = f(cosx)·(−sinx). The inner derivative (−sinx) is REQUIRED. Missing it costs most of the problem's points. This was 8 points on Exam 4.

Implicit Differentiation — Full Guide
When: Equation in x and y that can't easily be solved for y. Treat y as a function of x.

The rule: Differentiate both sides w.r.t. x. Every time you differentiate a term with y, apply chain rule: multiply by dy/dx.

d/dx(y²) = 2y·(dy/dx)
d/dx(y³) = 3y²·(dy/dx)
d/dx(sin y) = cosy·(dy/dx)
d/dx(xy) = y + x·(dy/dx) (product rule!)
d/dx(x²y) = 2xy + x²·(dy/dx)
ALGORITHM: Implicit Differentiation + Tangent Line
1Differentiate both sides of the equation w.r.t. x (chain rule on every y term)
2Collect all dy/dx terms on one side
3Factor out dy/dx: dy/dx = [stuff] / [stuff]
4Substitute the given point (x,y) to get the numerical slope
5Write tangent line using point-slope form

At (2,1): y′(6−2−2) = 1+4−1 → 2y′ = 4 → y′ = 2

Differentiability vs Continuity
Key theorem: If f is differentiable at a, then f is continuous at a.
Contrapositive: If f is NOT continuous at a, then f is NOT differentiable at a.
Converse is FALSE: f can be continuous at a but NOT differentiable. Example: f(x)=|x| at x=0 is continuous but not differentiable (corner).

Points of non-differentiability: Corners, cusps, vertical tangents, discontinuities.
FR Q32: Is f(x)={x² if x<1; 2x−1 if x≥1} differentiable at x=1?

Right derivative: lim(x→1⁺)[(2x−1)−1]/(x−1) = lim 2(x−1)/(x−1) = 2
Left = Right = 2 → differentiable at x=1, f′(1)=2
🔥 TRAP: (fg)′ = f′g′ is FALSE (T/F question Exam 2) (fg)′ = f′g + fg′ (product rule). The sum rule (f+g)′ = f′+g′ IS true. Also: d²y/dx² ≠ (dy/dx)² — the second derivative is NOT the square of the first derivative (another T/F trap from SG2).

#8 — Applications of Derivatives

A-TIER 85–90% probability · 6–14 pts
f′(x)→ Critical numbers→ Sign chart→ 1st deriv test→ Local extrema    f″(x)→ Concavity→ Inflection pts    EVT→ Abs max/min
Critical Numbers
c is a critical number of f if:
f′(c) = 0    OR    f′(c) is undefined AND c is in the domain of f.
🔥 TRAP (T/F Exam 3): Critical number ≠ local extremum These were T/F questions that caught many students:
• "If c is a critical number, then f′(c)=0." → FALSE (could be undefined)
• "If c is a critical number, then f(c) is a local extremum." → FALSE (could be inflection)
• "If f(c) is a local extremum, then c is a critical number." → TRUE
First Derivative Test
Setup: Find all critical numbers. Build a sign chart for f′.
• f′ changes − to + at c → local minimum
• f′ changes + to − at c → local maximum
• f′ doesn't change sign → neither (inflection point of f)

f increasing: f′ > 0     f decreasing: f′ < 0
Second Derivative Test
If f′(c) = 0:
• f″(c) > 0 → local minimum (concave up = "cup" = holds water ∪)
• f″(c) < 0 → local maximum (concave down = "cap" ∩)
• f″(c) = 0 → inconclusive → use first derivative test

Concave up: f″ > 0     Concave down: f″ < 0
Inflection Points
Inflection point = where concavity changes (f″ changes sign).
Candidate: f″(c) = 0 or f″(c) undefined Must verify: f″ actually changes sign at c. If f″(c)=0 but doesn't change sign → NOT an inflection point.
🔥 TRAP: f″(c)=0 alone is not enough Example: f(x) = x⁴. f″(x) = 12x². f″(0) = 0. But f″ doesn't change sign at x=0 (always ≥ 0). So x=0 is NOT an inflection point. Must check sign change.
Absolute Max/Min — Closed Interval Method (CIM)
EVT guarantees: If f continuous on [a,b], f MUST have an absolute max AND min on [a,b]. Always cite this before applying CIM.

Algorithm:
1. Find all critical numbers in interior (a,b) by solving f′(x)=0 and finding where f′ is undefined
2. Evaluate f at each critical number
3. Evaluate f(a) and f(b) (both endpoints)
4. Largest value = absolute max; smallest = absolute min
☠ EXAM KILLER: Always evaluate BOTH endpoints The absolute max/min is frequently at an endpoint. Students find the critical number values and forget to check f(a) and f(b). This costs the entire problem.

#9 — Curve Sketching — The 7-Step Algorithm

A-TIER 88% probability · 10–14 pts
THE 7 STEPS — use this exact order on every curve sketch problem:
1. DOMAIN — where is f(x) defined? Exclude zeros of denominator, negatives under even roots, etc.
2. INTERCEPTS — x-intercepts: set f(x)=0 and solve. y-intercept: compute f(0).
3. SYMMETRY — f(−x)=f(x) → even (y-axis symmetry). f(−x)=−f(x) → odd (origin symmetry).
4. ASYMPTOTES — Vertical: denominator=0 (and numerator≠0). Horizontal: lim(x→±∞). Oblique: if deg(num)=deg(den)+1, do polynomial long division.
5. f′(x) — Find critical numbers. Determine inc/dec intervals. Identify local maxima and minima (1st or 2nd deriv test).
6. f″(x) — Determine concavity (up/down). Find inflection point candidates, verify sign change.
7. SKETCH — Plot all key points, draw curve honoring asymptotes, extrema, concavity.
Reading f′(x) Graph to Describe f
Given a graph of y = f′(x):
• f′ > 0 (above x-axis) → f is increasing
• f′ < 0 (below x-axis) → f is decreasing
• f′ = 0 (x-intercept of f′) → critical number of f
• f′ changes + to − → local max of f
• f′ changes − to + → local min of f
• f′ is increasing → f is concave up
• f′ is decreasing → f is concave down
• f′ has local max/min → inflection point of f
🔥 TRAP: Confusing f and f′ on graph problems Exam 4 Q2 (6 pts) was given the graph of f′ and asked about f. The graph shows f′, NOT f. Local min of f′ = inflection point of f (NOT local extremum of f).
Oblique (Slant) Asymptotes
When: deg(numerator) = deg(denominator) + 1 exactly.
How: Polynomial long division: f(x) = Q(x) + R(x)/D(x)
As x→±∞: R(x)/D(x)→0, so oblique asymptote is y = Q(x).

Example: f(x) = (2x²+3x−5)/(x−1)
Divide: 2x+5 remainder 0/x−1 → y = 2x+5 is the oblique asymptote
EXAM STYLE — Full sketch: f(x) = 8/(x²+4)
1. Domain: All reals (denom x²+4 > 0 always) → (−∞,∞)
2. Intercepts: f(0) = 8/4 = 2. Set f=0: 8/(x²+4)=0 → no solution → no x-intercepts
3. Symmetry: f(−x) = 8/((−x)²+4) = 8/(x²+4) = f(x) → even (y-axis symmetric)
4. Asymptotes: No vertical (denom always > 0). lim(x→±∞) = 0 → y=0 horizontal
5. f′(x): = −16x/(x²+4)²
f′=0 → x=0. Sign chart: f′>0 for x<0 (increasing), f′<0 for x>0 (decreasing)
→ Local max at x=0: f(0) = 2
6. f″(x): = (48x²−64)/(x²+4)³
f″=0 → 48x²=64 → x²=4/3 → x=±2/√3
Check sign change at both → inflection points at x=±2/√3, f(±2/√3) = 8/(4/3+4) = 8/(16/3) = 3/2
7. Sketch: Bell-shaped curve, peak at (0,2), inflection at (±2/√3, 3/2), approaches y=0 asymptotically.

2. Intercepts: (0,0) only

Professor Traps Master List

☠ EXAM KILLER 1: Missing restriction on canceled factor When (x−a) cancels from num and denom: ALWAYS write "x≠a" in the answer. Example: "(x+1)/(x+2), x≠2." Professor deducts every time. Train yourself — write it automatically.
☠ EXAM KILLER 2: Not verifying theorem hypotheses For MVT, Rolle's, IVT, EVT, Squeeze: state the theorem, then explicitly verify each hypothesis. "f is a polynomial so it is continuous and differentiable everywhere" is acceptable. Skipping this step = large point loss on theorem problems.
☠ EXAM KILLER 3: Forgetting endpoints in CIM In the Closed Interval Method for absolute max/min, you MUST evaluate f at BOTH endpoints a and b as well as all critical numbers in (a,b). The absolute extremum is frequently at an endpoint. Missing endpoints = wrong answer.
🔥 TRAP 4: (fg)′ = f′g′ is FALSE The correct rule is (fg)′ = f′g + fg′. This was tested as a True/False question on Exam 2. The sum rule IS true: (f+g)′ = f′+g′. Also FALSE: d²y/dx² = (dy/dx)².
🔥 TRAP 5: Critical number ≠ local extremum (T/F Exam 3) "If c is a critical number, then f′(c)=0" → FALSE (f′(c) could be undefined). "If c is a critical number, f(c) is a local extremum" → FALSE (could be inflection). "If f(c) is a local extremum, then c is a critical number" → TRUE.
🔥 TRAP 6: Sign flip at x → −∞ with √ lim(x→−∞) x/√(4x²+1). Since x<0: √(x²)=|x|=−x. The limit is −1/2, NOT +1/2. This is a T/F-style calculation trap. Always note the sign when taking √(x²) for x→−∞.
🔥 TRAP 7: Implicit diff — missing dy/dx factor d/dx(y²) = 2y·(dy/dx). NOT just 2y. d/dx(sin y) = cosy·(dy/dx). Every y-term differentiation requires the chain rule multiplier dy/dx. This causes wrong tangent lines to implicit curves.
☠ EXAM KILLER 8: FTC2 chain rule missing on variable upper bound d/dx[∫ₐ^(cosx) f(t)dt] = f(cosx)·(−sinx). Students write only f(cosx). The inner derivative −sinx is required. This was worth 8 points on Exam 4. Train this reflex: whenever upper bound is g(x), multiply by g′(x).
🔥 TRAP 9: Related rates — substituting values before differentiating If you plug in the specific numerical values of variables before differentiating the equation, all terms with derivatives become zero. ALWAYS: differentiate the general equation FIRST, then substitute the instant values.
🔥 TRAP 10: Washer radii when rotating about y = k (not x-axis) Radii = distances from each curve to the axis y=k, NOT the y-values themselves. R = |f(x)−k|, r = |g(x)−k|. Draw a picture and measure the actual distances. Getting this wrong gives completely wrong volume.
🔥 TRAP 11: Inflection point requires f″ to CHANGE SIGN Setting f″(c)=0 only identifies candidates for inflection points. MUST verify that f″ actually changes sign at c (positive on one side, negative on the other). If f″(c)=0 but same sign on both sides → NOT an inflection point. Example: f(x)=x⁴ has f″(0)=0 but no inflection at x=0.
⚠ TRAP 12: Area between curves — must find intersections first Limits of integration are the intersection points of f and g. Setting up ∫₀⁵ without checking whether the curves cross in (0,5) gives wrong answer. Always: set f=g, solve for x, those are your bounds (or sub-bounds).
⚠ TRAP 13: Missing +C on indefinite integrals Every indefinite integral requires +C. Professor sees this as a notation/format error and deducts consistently. Definite integrals (with bounds) do NOT need +C. Build the habit: if no bounds, write +C.
